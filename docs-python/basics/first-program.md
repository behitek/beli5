# 🐍 Chương trình Python đầu tiên của bạn

## 🎯 Bạn sẽ học được gì?

Trong bài học này, bạn sẽ học cách tạo chương trình Python đầu tiên của mình! Giống như khi bạn dạy con rắn thông minh đầu tiên nói "Xin chào", đây là bước đầu tiên trong hành trình lập trình của bạn.

:::tip 💡 Gợi ý
Python 3.13 (phiên bản mới nhất năm 2025) có giao diện tương tác mới đẹp hơn với màu sắc rực rỡ và dễ sử dụng hơn bao giờ hết!
:::

## 🤔 Tại sao "Hello, World!" lại quan trọng?

Khi bạn học nói tiếng Việt, từ đầu tiên bạn học có thể là "Mama" hoặc "Papa". Trong lập trình, từ đầu tiên chúng ta dạy máy tính nói là "Hello, World!" (Xin chào, Thế giới!).

### 🐍 Ví dụ với con rắn thông minh
Hãy tưởng tượng bạn có một con rắn thông minh tên là Python. Bạn muốn dạy nó nói câu đầu tiên. Bạn sẽ:
1. 📝 Viết lời cho con rắn nhớ
2. 🗣️ Bảo con rắn nói ra
3. 👂 Nghe con rắn nói

## 💻 Chuẩn bị máy tính

### Cách 1: Sử dụng Python trên máy tính (Khuyên dùng)

Nếu bạn đã cài Python 3.13, hãy mở **Command Prompt** (Windows) hoặc **Terminal** (Mac/Linux):

```bash
python
```

```

Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

:::info 🌟 Tính năng mới Python 3.13
Giao diện tương tác mới có:
- 🎨 Màu sắc đẹp mắt
- ↩️ Chỉnh sửa nhiều dòng dễ dàng
- 📚 Lịch sử lệnh thông minh
- 🔍 Tự động hoàn thành code
:::

### Cách 2: Sử dụng trực tuyến (Dễ dàng hơn)

## 🎯 Viết chương trình đầu tiên

### Bước 1: Dạy con rắn nói "Xin chào"

Gõ dòng code này và nhấn Enter:

```python
print("Xin chào, Thế giới!")
```

🎉 **Kết quả:**
```
Xin chào, Thế giới!
```

### Bước 2: Hiểu code của bạn

Hãy phân tích từng phần:

```python
print("Xin chào, Thế giới!")
#  ↑        ↑
#  |        └── Nội dung muốn nói (trong dấu ngoặc kép)
#  └── Lệnh bảo Python nói ra
```

- `print` = "Hãy nói ra" (tiếng Anh: in ra)
- `"..."` = Nội dung cần nói (phải có dấu ngoặc kép)
- `()` = Nơi chứa thông tin cho lệnh

## 🎨 Thử nghiệm vui vẻ

### Thay đổi lời chào

```python
print("Xin chào, tôi tên là Minh!")
print("Hôm nay trời đẹp quá!")
print("Tôi đang học Python! 🐍")
```

### Sử dụng emoji và ký tự Việt

```python
print("🐍 Python rất thú vị!")
print("Tôi yêu lập trình 💻")
print("Việt Nam 🇻🇳 - Quê hương tôi")
```

### Nhiều dòng cùng lúc

```python
print("Dòng thứ nhất")

print("Dòng thứ ba")
```

**Kết quả:**
```
Dòng thứ nhất
Dòng thứ hai
Dòng thứ ba
```

## 🔧 Tạo file chương trình

### Bước 1: Tạo file mới

1. Mở **Notepad** (Windows) hoặc **TextEdit** (Mac)
2. Gõ code của bạn:

```python
# Chương trình Python đầu tiên của tôi
print("🐍 Xin chào! Tôi là Python!")
print("Tôi sẽ giúp bạn học lập trình")
print("Hãy cùng nhau khám phá thế giới code nhé!")
```

3. Lưu file với tên `chuong_trinh_dau_tien.py`

:::warning ⚠️ Lưu ý quan trọng
- File phải có đuôi `.py`
- Tên file không được có dấu cách (dùng dấu gạch dưới `_`)
- Tên file nên viết bằng tiếng Anh hoặc không dấu
:::

### Bước 2: Chạy file

Mở Command Prompt/Terminal, di chuyển đến thư mục chứa file và gõ:

```bash
python chuong_trinh_dau_tien.py
```

## 🎯 Thử thách vui

### Thử thách 1: Tự giới thiệu
Viết chương trình giới thiệu bản thân với:
- Tên của bạn
- Tuổi của bạn  
- Sở thích của bạn

```python
# Ví dụ mẫu
print("Xin chào! Tôi tên là Lan")
print("Tôi 12 tuổi")
print("Tôi thích đọc sách và chơi game")
```

### Thử thách 2: Tạo thiệp chúc mừng
Tạo một thiệp chúc mừng sinh nhật bằng Python:

```python
print("🎉🎂🎉🎂🎉🎂🎉")
print("  CHÚC MỪNG SINH NHẬT!")
print("🎁 Chúc bạn luôn vui vẻ 🎁")
print("🎉🎂🎉🎂🎉🎂🎉")
```

### Thử thách 3: Thơ bằng code
Viết một bài thơ ngắn bằng Python:

```python
print("🌸 Mùa xuân đến rồi")
print("🌺 Hoa nở khắp nơi")  
print("🦋 Bướm bay lượn lờ")
print("🌱 Cây xanh sum suê")
```

## ❗ Lỗi thường gặp và cách khắc phục

### Lỗi 1: Quên dấu ngoặc kép
❌ **Sai:**
```python
print(Xin chào)
```

✅ **Đúng:**
```python
print("Xin chào")
```

### Lỗi 2: Quên dấu ngoặc đơn
❌ **Sai:**
```python
print "Xin chào"
```

✅ **Đúng:**
```python
print("Xin chào")
```

### Lỗi 3: Viết sai lệnh
❌ **Sai:**
```python
Print("Xin chào")  # P viết hoa
```

✅ **Đúng:**
```python
print("Xin chào")  # p viết thường
```
